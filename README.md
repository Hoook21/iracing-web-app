# iRacing Web App

Projektgrundlage für persönliche iRacing-/Pitwall-Tools.

## Ziel

Eine Web-App für Rennvorbereitung, Strategie, Setup-Notizen, Rundenanalyse oder Rennlog entwickeln.

## Aktueller Stand

Dieses Repository enthält den vorhandenen Prototyp/Startpunkt. Weitere Anforderungen und konkrete Funktionen werden schrittweise ergänzt.

## Dateien

- `index.html` – aktueller App-/Prototyp-Stand

## Hinweis

Private Notizen, lokale Projektplanung und persönliche Renn-/Setup-Daten gehören in den lokalen Projektordner und nicht direkt in dieses öffentliche Repository.
